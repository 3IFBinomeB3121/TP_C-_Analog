NOM
       analog - Analyse un fichier de log Apache


SYNOPSIS
       ./analog [-g nomfichier.dot] [-e] [-t heure] nomFichier.log


DESCRIPTION
       analog  traite un fichier de log Apache (nomFichier.log), affiche les
       statistiques des opérations effectuées sur un serveur (nombre de consultations
       par url). Selon les options, il est possible de filtrer les données 
       ou/et de créer un fichier .dot qui sert de support à l'outil GraphViz.

       Par défaut (sans option) : affiche sur la console la liste des 10 urls de doc-
       uments les plus consultées par ordre dÃ©croissant
